Estadísticas Web

Estadísticas persistentes que se almacenan localmente

Información

Esta página web almacena estadísticas de uso de forma persistente en tu navegador. Cada vez que realizas una acción, se incrementa la estadística correspondiente y se guarda automáticamente.

  • Visitas a la web: Se incrementa cada vez que haces clic en "Simular Visita Web".
  • Clics en la página: Se incrementa cada vez que haces clic en "Simular Clic".
  • Tiempo navegación (min): Se incrementa cada vez que haces clic en "Simular Tiempo Navegación" (añade 5 minutos).
  • Páginas visitadas: Se incrementa cada vez que simulas una visita web.
  • Usuarios únicos: Se calcula en base a las visitas (aprox. 30% de las visitas totales).

Nota: Las estadísticas se guardan en el almacenamiento local de tu navegador. Si limpias los datos del navegador, se perderán las estadísticas.
